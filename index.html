เบื้องหลังเกิดอะไรขึ้น ทุกครั้งที่คุณเปิดเว็บไซต์?
เกี่ยวกับ RONGROI

RongRoi (ร่องรอย) เป็นโปรเจกต์โอเพนซอร์ส 100% ที่สร้างขึ้นเพื่อให้คุณเห็นชัด ๆ ว่าทุกครั้งที่คลิกลิงก์ คุณปล่อยข้อมูลรั่วไหลออกไปมากแค่ไหน เราไม่ได้พยายามขาย VPN ให้คุณ และเราไม่เก็บข้อมูลของคุณ เราแค่อยากให้คุณเห็นความจริงของการเก็บลายนิ้วมือเบราว์เซอร์ (browser fingerprinting) ในยุคนี้ ซอร์สโค้ดทั้งหมดเปิดให้ทุกคนตรวจสอบได้
ข้อควรทราบด้านความปลอดภัย

เมื่อคุณกดปุ่มด้านล่าง แสดงว่าคุณอนุญาตให้สคริปต์นี้เข้าไปอ่าน API ต่าง ๆ ในเบราว์เซอร์ของคุณ มันจะดึงข้อมูลตัวตนของฮาร์ดแวร์ ร่องรอยเครือข่าย และอะไรก็ตามที่ดึงได้ เพื่อแสดงให้เห็นว่า tracker ที่มองไม่เห็นมองเห็นอะไรจากคุณในทุก ๆ วัน

คำรับรองด้านความเป็นส่วนตัว: หน้านี้ไม่มีฐานข้อมูลเบื้องหลังเชื่อมต่ออยู่ ข้อมูลถูกดึงในเครื่องของคุณเอง และถูกลบทิ้งทันทีที่คุณปิดแท็บนี้
rongroi / เชลล์วิเคราะห์ร่องรอย

ชันสูตรร่องรอย: เราได้ข้อมูลนี้มาอย่างไร และจะหยุดมันได้อย่างไร

กำแพงข้อความที่เพิ่งไหลผ่านหน้าจอคุณไปเมื่อกี้? นั่นแหละคือร่องรอยดิจิทัลของคุณ และพูดตรง ๆ ว่ามันคือหายนะด้านความเป็นส่วนตัว เวลาคุณท่องเว็บ คุณไม่ได้แค่ขอหน้าเว็บจากเซิร์ฟเวอร์อย่างสุภาพ แต่เบราว์เซอร์ของคุณแทบจะยื่นแฟ้มประวัติเครื่องของคุณให้เขาแบบละเอียดยิบเป็นการแลกเปลี่ยน ทุกวันนี้เครือข่ายโฆษณาและนายหน้าค้าข้อมูลแทบไม่สนใจคุกกี้ติดตามแล้ว นั่นมันของเก่า แต่พวกเขาดูที่การผสมกันแบบเฉพาะเจาะจงสุด ๆ ของฮาร์ดแวร์ การตั้งค่าเครือข่าย และการตั้งค่าระบบที่คุณเพิ่งเปิดเผยออกไป เมื่อรวมกันแล้วมันกลายเป็น "ลายนิ้วมือ" ที่แทบจะไม่ซ้ำกับใครเลย

นี่คือรายละเอียดว่าสคริปต์นี้หลอกเบราว์เซอร์ของคุณให้คายข้อมูลออกมาได้อย่างไร และที่สำคัญกว่านั้น คุณจะปิดกั้นมันได้อย่างไร

1. การตรวจจับ VPN / Proxy

เราได้มาอย่างไร: การจับ VPN จากเบราว์เซอร์อย่างเดียวไม่ใช่เรื่องง่าย แต่ระบบของคุณจะแฉตัวเองถ้าไม่ระวัง เราตรวจโซนเวลาของระบบปฏิบัติการของคุณ (ผ่านนาฬิการะบบในเบราว์เซอร์) แล้วนำไปเทียบกับโซนเวลาตามภูมิศาสตร์ของ IP คุณ ถ้าคุณนั่งอยู่ที่กรุงเทพฯ แต่ IP บอกว่าคุณอยู่นิวยอร์ก นาฬิกาจะไม่ตรงกัน คุณจะถูกตั้งธงทันที นอกจากนี้เรายังเทียบผู้ให้บริการอินเทอร์เน็ต (ISP) ของคุณกับรายชื่อโฮสต์ศูนย์ข้อมูล (เช่น AWS หรือ DigitalOcean) เพราะคนทั่วไปไม่นั่งดู YouTube จากฟาร์มเซิร์ฟเวอร์

วิธีป้องกัน: ปลอมโซนเวลาของระบบให้ตรงกับเซิร์ฟเวอร์ VPN ของคุณ ส่วนขยายอย่าง Spoof Timezone ช่วยทำอัตโนมัติได้ ส่วนการเลี่ยงการตรวจศูนย์ข้อมูล คุณต้องใช้ Residential Proxy ที่ส่งทราฟฟิกผ่านการเชื่อมต่อของบ้านเรือนจริง ๆ

2. Canvas Fingerprint

เราได้มาอย่างไร: HTML5 มีเครื่องมือวาดภาพชื่อ <canvas> เว็บไซต์ใช้มันทำเกมและกราฟ ในสคริปต์นี้เราสร้าง canvas ที่มองไม่เห็นขึ้นมา แล้วบังคับให้เบราว์เซอร์ของคุณวาดภาพเฉพาะแบบซ้อนชั้นด้วยสีและฟอนต์ต่าง ๆ เพราะคอมพิวเตอร์แต่ละเครื่องมีการผสมของไดรเวอร์กราฟิก ฟอนต์ และอัลกอริทึม sub-pixel smoothing ที่ต่างกันเล็กน้อย ภาพที่ออกมาจึงไม่ซ้ำกันในระดับจุลภาค เราถ่ายภาพพิกเซลที่มองไม่เห็นเหล่านั้นแล้วแปลงเป็นรหัส ID

วิธีป้องกัน: ตัวบล็อกโฆษณาพื้นฐานทำอะไรกับเรื่องนี้ไม่ได้เลย เพราะการวาด canvas เป็นฟีเจอร์หลักของเว็บ คุณต้องใช้เบราว์เซอร์ที่ใส่ใจเรื่องนี้จริง ๆ อย่าง Brave หรือ Tor Browser ผู้ใช้ Firefox ติดตั้ง CanvasBlocker ได้ ซึ่งจะแทรก "สัญญาณรบกวน" แบบสุ่มเข้าไปในการคำนวณทุกครั้งที่เว็บพยายามวาดภาพซ่อน ทำลายการติดตามของพวกเขาจนหมด

3. AudioContext Fingerprint

เราได้มาอย่างไร: เราใช้ Web Audio API สร้างคลื่นเสียง sine ความถี่สูง ส่งผ่าน compressor เสียงดิจิทัล แล้ววัดผลลัพธ์ เราไม่ได้เปิดเสียงผ่านลำโพงของคุณด้วยซ้ำ แค่จับบัฟเฟอร์เสียงภายในมา ฮาร์ดแวร์การ์ดเสียงเฉพาะของคุณกำหนดว่าการคำนวณทศนิยมนั้นจะถูกประมวลผลอย่างไรพอดี เป็น ID ฮาร์ดแวร์ที่แม่นยำและแนบเนียนอย่างน่าตกใจ

วิธีป้องกัน: คล้ายกับการติดตาม canvas คุณกำลังสู้กับเครื่องมือหลักของเว็บ การใช้เบราว์เซอร์ Brave เป็นทางเลือกที่ดีที่สุด มันดักคำขอ audio API และแทรกสัญญาณรบกวนเข้าไปในผลลัพธ์ ทำให้อัลกอริทึมของพวกเขาพัง

4. การดึงข้อมูล WebGL และ GPU

เราได้มาอย่างไร: WebGL คือเอนจินที่เบราว์เซอร์ใช้เรนเดอร์กราฟิก 3 มิติ เราเปิดสภาพแวดล้อม 3 มิติขึ้นในเบื้องหลัง แล้วขอชื่อฮาร์ดแวร์ที่ทำงานหนักผ่านส่วนขยายดีบัก (WEBGL_debug_renderer_info) เบราว์เซอร์ของคุณก็ยินดียื่นยี่ห้อและรุ่นการ์ดจอที่แน่นอนให้ ง่ายแค่นั้นเอง

วิธีป้องกัน: คุณสามารถปิด WebGL ทั้งหมดได้ในการตั้งค่าขั้นสูงของเบราว์เซอร์ (ใน Firefox พิมพ์ about:config แล้วตั้ง webgl.disabled เป็น true) แต่จำไว้ว่ามันจะทำให้เกม 3 มิติในเบราว์เซอร์และ Google Earth ใช้ไม่ได้

5. การนับอุปกรณ์สื่อแบบเงียบ ๆ

เราได้มาอย่างไร: รู้ไหมว่าเว็บไซต์ต้องขออนุญาตก่อนใช้กล้องและไมค์ของคุณ? ใช่ แต่พวกเขาไม่ต้องขออนุญาตเพื่อ นับ จำนวน โดยการเรียก navigator.mediaDevices.enumerateDevices() เราถามเบราว์เซอร์ของคุณว่ามีอินพุตเสียงและเลนส์กี่ตัวเสียบอยู่กับเครื่อง แม้ไม่มีสิทธิ์บันทึก แต่การรู้ว่าคุณมีกล้องสองตัวและอินพุตเสียงสามตัวก็เพิ่มจุดข้อมูลเฉพาะให้โปรไฟล์การติดตามของคุณ

วิธีป้องกัน: Safari จำกัดเรื่องนี้ไว้ตั้งแต่ต้นซึ่งถือเป็นข้อดีที่หายากของ Apple ส่วน Chrome และ Firefox คุณต้องเข้าไปที่แฟล็กการตั้งค่าขั้นสูงเพื่อปลอมแปลง หรือพูดง่าย ๆ ก็แค่ถอดไมค์และกล้องภายนอกออกเมื่อไม่ได้ใช้

6. ข้อมูลฮาร์ดแวร์และเครือข่าย

เราได้มาอย่างไร: ส่วนนี้เราไม่ต้องแฮกอะไรเลย เบราว์เซอร์ของคุณแพร่ข้อมูลออกมาเองแบบไม่คิด ด้วยคำสั่งมาตรฐาน เราถามเครื่องคุณว่ามี CPU กี่คอร์ ติดตั้ง RAM เท่าไร และความลึกสี (color depth) ของหน้าจอ เมื่อรวมกับ IP ข้อมูลเหล่านี้ก็เพียงพอจะติดตามคุณข้ามเว็บไซต์ได้แม้คุณจะล้างคุกกี้ทุกสิบนาที

วิธีป้องกัน: การป้องกันที่ดีที่สุดคือการกลมกลืนไปกับคนอื่น Tor Browser จัดการเรื่องนี้ด้วยการบังคับให้ผู้ใช้ทุกคนใช้ขนาดหน้าจอ โปรไฟล์ฮาร์ดแวร์ และโซนเวลาเดียวกันหมด ทำให้แยกคุณออกจากคนอื่นไม่ได้ สำหรับการใช้งานประจำวัน เบราว์เซอร์ที่เน้นความเป็นส่วนตัวควบคู่กับ VPN ที่เชื่อถือได้คือทางที่ดีที่สุดในการหยุดการรั่วไหล

7. คะแนนความปลอดภัยของเบราว์เซอร์

ทำงานอย่างไร: หลังการตรวจสอบทุกครั้ง RongRoi จะให้เกรดสถานะความเป็นส่วนตัวของเบราว์เซอร์คุณในระดับ 0 ถึง 10 คะแนนคำนวณจากสัญญาณสิบเอ็ดอย่างที่เก็บระหว่างสแกน เช่น มีตัวบล็อกโฆษณาทำงานอยู่ไหม การเก็บลายนิ้วมือ canvas และ audio ถูกบล็อกไหม ตัวตน GPU ถูกปิดบังไหม ใช้ VPN อยู่ไหม มีการส่งเฮดเดอร์ DNT และ Global Privacy Control ไหม และอื่น ๆ แต่ละสัญญาณมีค่าน้ำหนักคะแนน แล้วรวมเป็นคะแนนเต็ม /10 พร้อมเกรดตัวอักษร

วิธีเพิ่มคะแนน: รายละเอียดในเทอร์มินัลจะบอกชัดว่าการตรวจสอบข้อไหนไม่ผ่าน วิธีที่ได้ผลเร็วที่สุดคือ ติดตั้งตัวบล็อกโฆษณาอย่าง uBlock Origin เปลี่ยนไปใช้ Brave (บล็อกลายนิ้วมือ canvas, audio และ WebGL ให้ตั้งแต่ต้น) และเปิด VPN การเปิด Do Not Track และ Global Privacy Control ในการตั้งค่าเบราว์เซอร์นั้นง่ายและฟรี ส่วนการปิดบังข้อมูลฮาร์ดแวร์ (จำนวนคอร์ CPU, RAM) ต้องใช้เบราว์เซอร์ที่เน้นความเป็นส่วนตัวหรือส่วนขยายอย่าง CanvasBlocker บน Firefox
